BASIC(1964)

BASIC is one of earliest programming languages published in 1964. It was developed by John G. Kemeny, Thomas E. Kurtz and Mary Kenneth Keller at Dartmouth College. BASIC stands for "Beginner's All-purpose Symbolic Instruction Code." It was designed to allow students at Dartmouth College to write mainframe computer programs for their Time-Sharing System. It concentrated on mathematical work containing arithmetic functions inside of the program.

Pascal(1970)

Pascal is a procedural programming language designed by Niklaus Wirth that was published in 1970. It was based on a concept called Structure Programming that had become recently popular. It contained data structures such as lists and graphs. Its purpose was to encourage good programming practices by using Structure Programming and to use dynamic and recursive data structures.

C++(1985)

C++ was designed by Bjarne Stroustrup in 1985. He had begun to work with the predecessor of C++, C. He created a new version called “C with Classes” that added new features such as virtual functions, type-safe free-store memory allocation, improved type checking, and single-line comments with two forward slashes. Eventually “C with Classes” was renamed to C++.
