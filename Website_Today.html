Java Script

Java Script is a more modern programming language used in various websites and webpages. It is known for not being too hard to learn and is also known for it easy access and flexibility. Heck, the website your on right now uses Java Script!! There is not one website on the web that doesn't use Java Script. Don't mistaken Java Script for Java, Java is another language in its self, it is mainly used for created programs, meanwhile Java Script is used for creating programs that run on a browser.

Python

Python is known for its readability and the access to many libraries, for example, there are even game libraries! Python is a good method of introducing new programmers into programming. The creator of python was Guido van Rossum, Python used to be an idea, but obviously, is a reality now. There were various versions for python, there was 2.0 back in 2000, and now it is up to version 3.6.
